# nostr-nowplaying

Update #nowplaying NIP-38 status (kind 30315) for Rhythmbox

## Usage

```
$ ./nostr-nowplaying.sh algia
```
or
```
$ ./nostr-nowplaying.sh nostk
```

## Requirements

* [algia](https://github.com/mattn/algia) or [nostk](https://github.com/mitsugu/nostk)
